# 一个简单的记录版（类似Trello）
<!-- 上传git后，public／演示图。gif的地址 -->
![image](https://github.com/chentianwei411/a-simple-reminder/blob/master/public/%E6%BC%94%E7%A4%BA%E5%9B%BE.gif)


用途：记录日常信息。

* 新增管理list,及card。

* 可以拖拉移动，list, card

技术：
* Vue.js，使用单文件组件。使用了draggable插件。

* Vuex管理状态。

* 使用了ActionCable，可以跨浏览器同步网页。
